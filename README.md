## 功能特性

* **动态版本下载**：集成 GitHub API，网页端可自动识别并抓取最新发布的整合包（`.zip`）以及官方最新版本的 HMCL 启动器（`.exe`）。
* **自动化容错机制**：当 GitHub API 接口因网络波动请求失败时，系统将自动启用备用方案，引导玩家至官方分发渠道。
* **响应式体验**：优化了移动端与桌面端的排版展示，确保玩家在不同设备上均能获得流畅的阅读与下载体验。

---

## 二次开发主要修改点

1. **按钮逻辑重构**：将原有的静态下载链接升级为异步 JavaScript 请求，实现“一劳永逸”的无感更新。
2. **主题色系调整**：根据服务器风格，将下载引导按钮等核心交互元素的视觉样式调整为绿色主题（`#22c55e`）。
3. **文本定制**：修改了卡片描述语与步骤指引，使其完全贴合 Nika 服务器的玩家入服流程。
